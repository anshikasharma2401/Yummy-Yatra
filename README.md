# Yummy-Yatra-Project
A Recipe Management Web Application built using - JSP, Servlets , JDBC , Maven , Tomcat , HTML , CSS , Bootstrap & JavaScript !
This Porject allows users to Add, View and manage recipes in an easy and interactive way...
---
FEATURES 
## USER FEATURES ##
: Add New Recipes
: View Recipes
: Search Recipes 
: Read Full Recipes Details
: Responsive UI using Bootstrap

****Technical Features****
: Java EE (JSP + Servlet)
: JDBC Connectivity
: MSSQL Database
: MAven Project Structure
: MVC Architecture
: Deployed on Apache Tomcat

****Tech Stack****

| Category | Technologies |
|---------|--------------|
| **Frontend** | HTML, CSS, Bootstrap, JavaScript |
| **Backend** | JSP, Servlets (Java) |
| **Database** | MS SQL Server |
| **Server** | Apache Tomcat |
| **Build Tool** | Maven |
| **IDE Used** | NetBeans |
